Ordered list

  1. First do this
  2. Then do that
  3. Next, do this other thing
  4. Fourth, bake at 350 degrees for an hour
  5. Finally we all head over to a party at Warren's

Ordered lists also have attributes:

  • START (to set starting value)
  • REVERSE (to reverse the sort oder)
  • VALUE (to change the value assigned to an list item in the list)

Nested List

where an ordered or unordered list is added as a list item (LI) to a higher level UL or OL. List item MUST follow OL or UL, but can contain its own OL or UL element.

  • First do this
  • Then do that
  • Next, do these other things:
    • First Item
    • Second Item
    • Third Item
    • Fourth Item
  • Fourth, set on fire then wait to cool
  • Finally, Frank said there is a party at Warren's
